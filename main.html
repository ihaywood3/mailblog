Email-based Blogging

This is a basic blogging platform inspired by Hey, World

To post, email world@haywood.id.au. There is no need to register, an account will be created the first time you send an email.

If you send an HTML email, that will be used. If plain text, it will be interpreted as Markdown.

Images are not supported. If they are in the future, registration/payment of some sort will be required.
